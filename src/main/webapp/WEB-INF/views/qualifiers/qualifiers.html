The specification for CDI Qualifiers

Usually in CDI we inject the interface type of a CDI Bean and not the implementation type, so we need to tell CDI what implementation instance we want to have injected if there are multiple implementations available. CDI uses type safe qualifiers which are implemented as annotations for the selection of the implementation. EJBs for instance use string names for the implementation selection which is error prone and not type safe.
